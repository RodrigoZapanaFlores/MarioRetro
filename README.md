# Mario Retro
## Descripción
Juego sencillo en 2D donde controlas a Mario que puede saltar sobre enemigos y moverse por el escenario.  
El fondo se mueve de forma continua, simulando un "scroll" lateral, y Mario tiene animación de caminar.

## Tecnologías
- HTML5 Canvas
- CSS3
- JavaScript 

## Cómo jugar
1. Abrir `index.html` en un navegador moderno.
2. Usar las flechas del teclado:
   - Saltar
   - Mover a la derecha
   - Mover a la izquierda
3. Pulsar el botón PLAY para iniciar y PAUSA para detener.

## Archivos
- index.html → Estructura del juego
- css/index.css` → Estilos
- js/ → Scripts del juego (player, background, game, etc.)
- imagenes/ → Sprites de Mario y el fondo
- audio/ → Efectos de sonido y música
